# 3.2.15


## 解答

比较简单，这里比较/取值都算一次访问，因此 `keys` 的访问序列会出现重复元素。

| 函数       | 路径                    |
| ---------- | ----------------------- |
| floor(Q)   | E Q                     |
| select(5)  | E Q                     |
| ceiling(Q) | E Q                     |
| rank(J)    | E Q J                   |
| size(D, T) | E Q T E D               |
| keys(D, T) | E D E Q J J M Q T S S T |
