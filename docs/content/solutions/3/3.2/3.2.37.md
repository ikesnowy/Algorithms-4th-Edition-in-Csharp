# 3.2.37


## 解答

二叉树层序遍历，出队一个结点，打印它，将结点的左右子树入队，循环即可。

## 代码

```csharp
private void PrintLevel(Node x)
{
    var queue = new Queue<Node>();
    queue.Enqueue(x);
    while (queue.Count > 0)
    {
        var node = queue.Dequeue();
        if (node.Left != null)
            queue.Enqueue(node.Left);
        if (node.Right != null)
            queue.Enqueue(node.Right);
        Console.Write(node.Key + ", ");
    }
}
```
