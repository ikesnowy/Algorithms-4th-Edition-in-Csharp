# 1.1.14


## 解答

简单使用 log 的定义逼近即可。

## 代码

```csharp
static void Main(string[] args)
{
    int N = 9;
    Console.WriteLine($"{ lg(N)}");
}

//利用循环逼近 N，得到 log2(N) 的值
static int lg(int N)
{
    int baseNumber = 2;
    int pow = 1;
    int sum = 2;

    for (pow = 1; sum < N; ++pow)
    {
        sum *= baseNumber;
    }

    return pow - 1;
}
```
