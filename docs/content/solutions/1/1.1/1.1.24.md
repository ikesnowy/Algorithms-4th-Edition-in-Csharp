# 1.1.24


## 解答

在书本中 GCD 的基础上，在函数开始时增加一条输出语句即可。

## 代码

```csharp
static void Main(string[] args)
{
    GCD(105, 24);
    Console.WriteLine();
    GCD(111111, 1234567);
}

public static int GCD(int a, int b)
{
    Console.WriteLine($"{a} {b}");
    if (b == 0)
    {
        return a;
    }

    return GCD(b, a % b);
}
```
