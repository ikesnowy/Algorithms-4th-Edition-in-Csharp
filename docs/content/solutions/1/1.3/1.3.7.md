# 1.3.7


## 解答

链表实现的话就是返回第一个结点 first 的 item 字段。

数组实现的话就是返回 first 对应的数组元素。

这里给出链表实现，完整实现见习题 1.3.2 的代码。

## 代码

```csharp
/// <summary>
/// 返回栈顶元素（但不弹出它）。
/// </summary>
/// <returns></returns>
public Item Peek()
{
    if (IsEmpty())
        throw new InvalidOperationException("Stack Underflow");
    return this.first.item;
}
```
