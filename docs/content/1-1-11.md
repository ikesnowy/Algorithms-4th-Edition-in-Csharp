# 1.1.11


## 解答

注意，二维数组 bool[M, N] 代表 M 行 N 列的布尔数组。
使用二重循环即可实现。
输出使用制表符 ’\t’ 作为分隔。

## 代码

```csharp
static void PrintArray2D(bool[,] array)
{
    int rows = array.GetLength(0);//获取行数
    int columns = array.GetLength(1);//获取列数

    //输出列号
    for (int i = 0; i < columns; i++)
    {
        Console.Write($"\t{i + 1}");
    }

    Console.Write("\n");

    for (int i = 0; i < rows; i++)
    {
        //输出行号
        Console.Write($"{i + 1}");
        for (int j = 0; j < columns; j++)
        {
            if (array[i, j])
            {
                Console.Write($"\t*");
            }
            else
            {
                Console.Write($"\t ");
            }
        }
        Console.Write("\n");
    }
}
```
