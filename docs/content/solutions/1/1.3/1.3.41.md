# 1.3.41


## 解答

可以按照书上的提示出队再入队，也可以直接用迭代器访问一遍进行复制。

## 代码

```csharp
/// <summary>
/// 复制构造函数。
/// </summary>
/// <param name="r"></param>
public Queue(Queue<Item> r)
{
    foreach (Item i in r)
    {
        Enqueue(i);
    }
}
```
