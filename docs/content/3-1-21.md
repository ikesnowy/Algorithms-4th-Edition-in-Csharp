# 3.1.21


## 解答

`BinarySearchST`
包含一个键数组和一个值数组，以及一个 `int` 变量。
数组长度变化范围为 N~4N ，故总大小：
从 2 × (24 + 8N) +4 = 52 + 16N 字节 （100 %），
到 2 × (24 + 32N) +4 = 52 + 64N 字节（25 %）之间变动。
`SequentialSearchST`
包含 N 个结点以及一个 `int` 变量
(16 + 8 + 8 + 8)N + 4 = 4 + 40N 字节
