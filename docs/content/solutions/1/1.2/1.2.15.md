# 1.2.15


## 解答

这里我们基于 File.ReadAllLines() 进行实现。

## 代码

```csharp
public static int[] ReadInts(string path)
{
    string[] allLines = File.ReadAllLines(path);
    int[] result = new int[allLines.Length];

    for (int i = 0; i < allLines.Length; ++i)
    {
        result[i] = int.Parse(allLines[i]);
    }

    return result;
}
```
